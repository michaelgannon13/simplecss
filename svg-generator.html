Simple CSS SVG

Simple CSS SVG is a free css generator for Web Designers & Front End Developers to simplify creating SVG shapes by providing a 'jumping off point'. Just click the SVG to generate the SVG you want. If you want to change the paths, colors etc you already have the template of the shape you want so all that needs to be done is to modify it.
