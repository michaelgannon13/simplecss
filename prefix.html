Simple CSS Vendor Prefixes

Simple CSS Vendor Prefixes is a free tool for Web Designers & Front End Developers to make css vendor prefixes, simple and easier to work with. Once your site is supported on every screen it’s time to support it on every browser! Just type in the CSS property you want to prefix into the search bar and click the result to generate your vendor prefix for the appropriate browsers. If you have any feedback please let us know.
